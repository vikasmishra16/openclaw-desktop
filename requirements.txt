# ============================================================
# OpenClaw Desktop — Dependency Requirements
# ============================================================
# This project uses npm (Node.js) and Cargo (Rust) for
# dependency management. Run the install commands below.
# ============================================================

# ==================== SYSTEM PREREQUISITES ====================
# Install these before setting up the project:
#
# 1. Node.js >= 18.x          https://nodejs.org/
# 2. Rust (stable toolchain)  https://rustup.rs/
# 3. Ollama (local LLM)       https://ollama.ai/
#    Run: ollama pull llama3
# 4. Google Chrome             (for Playwright browser automation)

# ==================== NODE.JS DEPENDENCIES ====================
# Install with: npm install
